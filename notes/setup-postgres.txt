If you haven't used postgres in a while, recall that the 
service start goes like this:

  postgres -D $PGROOT/data/

While the interactive terminal (with superuser login) 
goes like this:

  psql -U postgres

The first thing you'll want to do is set a new superuser 
password:

  alter user postgres with password 'newpassword';
